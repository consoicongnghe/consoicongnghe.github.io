Các bước cài đặt MS Office của Sói

  1. Tải file cài đặt Office từ máy chủ Microsoft.
  2. Tải file cấu hình muốn cài đặt:
    • Toàn bộ bộ phần mềm: alloffice.xml
    • Chỉ Word, Excel và PowerPoint: only3.xml
  3. Copy 2 file vừa tải (setup.exe và *****.xml) vào ổ C
  4. Mở Command Prompt dưới quyền Administrator, chạy lệnh này:
    • Toàn bộ: cd /d C:\ && setup.exe /configure alloffice.xml
    • Chỉ 3 phần mềm: cd /d C:\ && setup.exe /configure only3.xml
