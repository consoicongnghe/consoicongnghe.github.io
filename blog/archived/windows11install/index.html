Các bước cài đặt Windows 11 của Sói

Backup:

  • Danh sách chương trình đã cài đặt trong Control Panel
  • Bookmark trong trình duyệt web (trừ khi đã bật đồng bộ hóa)
  • Kiểm tra Recycle Bin
  • Chạy
    • Export wifi
    • User folders
    • (Web browsers' user data, OBS, steam userdata, qbittorrent, TrafficMonitor)
    • Experimental: Desktop icon layout
CÀI ĐẶT WINDOWS BẰNG BỘ CÀI ISO
  • OOBE (lần đầu khởi động): Shift+F10 --> start ms-cxh:localonly
  • ngoài ra trên laptop nhớ
    1. Shift+F10 --> manage-bde -status
    2. Tắt Bitlocker: Shift+F10 --> regedit --> HKLM\SYSTEM\CurrentControlSet\Control\BitLocker
    3. Bấm chuột phải mục BitLocker, chọn New Dword (32-bit) value: PreventDeviceEncryption, value=1
    • Chú ý: Nếu đã vào Windows và Bitlocker ở trạng thái chờ Waiting for activation, mở Command Prompt bằng quyền Admin và gõ: manage-bde -off C:
HOÀN THÀNH OOBE, KHỞI ĐỘNG VÀO DESKTOP
  • Nếu sử dụng GPU NVIDIA 30xx-, cài driver GPU v566.36 trước. Các đời sau cài driver mới nhất trước.
  • Chạy
    • Chỉnh Terminal luôn chạy quyền admin
    • Đổi tên ổ C:/ thành Windows
    • Đổi tên PC nếu muốn (trong System Settings/System/About/
    • /Windows/RegTweaks/windows11_postInstall.reg
    • /Windows/PostInstall/RegTweaks/r1_script/1_guest.cmd
    • Cập nhật App Installer: /Windows/PostInstall/winget/offline/*
RESTART
  • Check ô Receive updates for other Microsoft products, Check for updates
  • Kết nối internet / import wifi
  • Chạy
    • Tắt Defender's Realtime virus protection
    • /Windows/PostInstall/winget/MsStore_UpdateAll.ps1 --> open MS Store
    • Cài đặt Unikey
    • Cài đặt Microsoft Media Extensions
    • Cài đặt Visual C++ Redistributable Runtimes: /Windows/Driver/VCRedist/install_all.bat
    • Restore User folder đã backup
    • Cài Apple ProRes RAW, cập nhật Terminal, cài MS .NET Desktop Runtime: /Windows/PostInstall/winget/1_guest.cmd
    • Cài đặt 7-zip, K-Lite Codec Pack, TrafficMonitor, RustDesk, custom cursor
    • Cài đặt Microsoft Office
    • Cài đặt font VN: /Windows/Driver/fonts/
    • Snipping Tool: /settings: tắt autosave, tắt multiple windows
    • activ win+office: irm https://get.activated.win | iex
    • Dọn sạch ổ C:/
    • Xóa C:\z\
  • Control Panel\Hardware and Sound\Power Options\Edit Plan Settings và \System Settings
  • (di chuyển vị trí folder ~/Downloads)
  • (/Programs/1sthing/, Chrome, Steam + /Programs/zInstallers/, hwinfo, notepad++)
RESTART
  • Tắt Defender's Realtime virus protection
  • Tối ưu Task Manager (tab Startup apps)
  • (setup personal programs, Adobe các thứ)
