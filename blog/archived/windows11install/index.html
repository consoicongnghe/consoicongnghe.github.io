Các bước cài đặt Windows 11 của Sói

Backup:

  • User folders
  • Program list in control panel
  • Backup browser bookmarks (unless synced)
  • Fonts
  • Check Recycle Bin
  • (OBS, steam userdata)
  • Export wifi
INSTALL

OOBE (first start setup): Shift+F10 --> OOBE\BYPASSNRO

also check Bitlocker: Shift+F10 --> manage-bde -status

Disable Bitlocker: Shift+F10 --> regedit --> HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\BitLocker. Right click BitLocker, New Dword (32-bit) value: PreventDeviceEncryption, value=1

FINISH OOBE
  • Check Windows edition, go Pro.
  • (Rename PC, C:/, change Download folder location)
  • Windows/PostInstall/GroupPolicy/ --> copy to C:/Windows/system32
  • Windows/PostInstall/RegTweak/ReclaimWindows/1_guest (+Unpin +UninsOnedrive +UninsMSapps)
  • Windows/RegTweaks/windows post-install
RESTART
  • virus protection: turn off, (add exception)
  • connect internet / import wifi
  • timezone, sync time
  • check for updates
  • Windows/Driver/fonts/
  • Windows/Driver/VCRedist/
  • Windows/PostInstall/winget/MicrosoftStoreAppUpdater.ps1 --> open MS Store
  • setup chrome, office, unikey, (adobe, trafficMonitor)
  • Programs/Chrome/ --> Ublock Origin, SponsorBlock, FastForward, Run In Background
  • Settings: (touchpad -> 3+4finger = middle click), language bar)
  • set wp and lock screen pic
  • setups/prog/1sthing/
  • setup things except k-lite, adobe.
  • activ win+office: irm https://get.activated.win | iex
  • Razer Synapse? Logi Options+?
RESTART
  • k-lite, adobe
  • Windows/PostInstall/winget/1_guest.cmd
  • post-update: control panel/power options
  • Settings: startup app
  • rustdesk: /security: enable remote cfg modification, set permanent pw
  • Snipping Tool settings: dont auto save, separate windows
  • Laptop: check modern standby status: powercfg /a. Enable/Disable by registry tweak.
  • cleanup C:/

Trường hợp cài gấp/công nghiệp: Chrome, Unikey, Office. 1_guest + post-install.reg. Crack.
