Các bước cài đặt Windows 11 của Sói

Backup:

  • Program list in control panel
  • Backup browser bookmarks (unless synced)
  • Check Recycle Bin
  • Run BackupPreInstall.cmd,
    • User folders
    • Fonts
    • (OBS, steam userdata)
    • Export wifi
INSTALL

OOBE (first start setup): Shift+F10 --> OOBE\BYPASSNRO

also .

  • Shift+F10 --> manage-bde -status
  • Disable Bitlocker: Shift+F10 --> regedit --> HKLM\SYSTEM\CurrentControlSet\Control\BitLocker
  • Right click BitLocker, New Dword (32-bit) value: PreventDeviceEncryption, value=1

FINISH OOBE
  • Check Windows edition, go Pro
  • (Rename PC, C:/, change Download folder location)
  • Run GuestSetup1.cmd,
    • /PostInstall/GroupPolicy/ --> copy to C:/Windows/system32
    • /PostInstall/RegTweaks/r1_script/1_guest.cmd
    • /RegTweaks/windows11_postInstall.reg
RESTART
  • connect internet / import wifi
  • auto timezone, toggle auto time sync
  • check for updates
  • /Driver/fonts/ --> install all
  • setup chrome (office, unikey, trafficMonitor, Logi Options+)
  • Programs/Chrome/extensions.bat: Ublock Origin, SponsorBlock, FastForward, Run In Background
  • set wp and lock screen pic
  • Run GuestSetup2.cmd,
    • Restore previously backed up user folders
    • /Driver/VCRedist/install_all.bat
    • /PostInstall/winget/MsStore_UpdateAll.ps1 --> open MS Store
    • (/Programs/1sthing/)
    • setup things except k-lite, adobe.
    • activ win+office: irm https://get.activated.win | iex
RESTART
  • virus protection: turn off, (add exception GG Drive)
  • k-lite, adobe
  • Settings: startup app
  • Laptop: check modern standby status: powercfg /a. Enable/Disable by registry tweak.
  • Run GuestSetup3.cmd,
    • /PostInstall/winget/1_guest.cmd
    • post-update: Control Panel/Power options
    • Rustdesk: /security: enable remote cfg modification, set permanent pw
    • Snipping Tool/Settings: dont auto save, multiple windows
    • Disk cleanup C:/

Trường hợp cài gấp/công nghiệp: Chrome, Unikey, Office. 1_guest + post-install.reg. Crack.
