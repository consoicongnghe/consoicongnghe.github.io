Virus tạo shortcut, ẩn file trong USB

Virus shortcut

Trong danh sách file trong USB có một file shortcut .lnk có tên giống tên USB hoặc ổ cứng trong máy. Cách cứu:

  1. Bấm vào thanh tìm kiếm, gõ *.exe và *.lnk
  2. Xóa toàn bộ các file .lnk và file .exe
  3. Copy toàn bộ dữ liệu khác ra ngoài
  4. Format USB
  5. Copy dữ liệu vào lại USB

USB có dữ liệu chiếm dung lượng nhưng mở ra không thấy file đâu

  1. Mở Command Prompt quyền admin, gõ lệnh attrib -h -r -s /s /d X:\*.* (X là tên ổ USB, ví dụ: E, F,...)
  2. Bấm vào thanh tìm kiếm, gõ *.exe, xoá toàn bộ các file .exe
  3. Copy toàn bộ dữ liệu ra ngoài
  4. Format USB
  5. Copy dữ liệu vào lại USB
