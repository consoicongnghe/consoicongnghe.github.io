Các bước kiểm tra khi mua laptop

  • Vỏ máy, mặt màn hình (scratch?)
  • Màn hình: điểm chết, response time, độ phân giải, độ sáng
  • Bàn phím: en.key-test.ru
  • Touchpad: (force touch?)
  • Test các cổng USB, khe đọc thẻ nhớ, cổng 3.5mm v.v.
  • Webcam, microphone, loa
  • Ổ cứng (CrystalDisk)
  • Cắm sạc: lỏng hay không
  • Wifi, Bluetooth
    • Macbook specific: cycle count
    • Windows specific: powercfg /batteryreport /output "%USERPROFILE%/Desktop/batt.html"
