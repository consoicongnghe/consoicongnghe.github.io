Format USB định dạng FAT32

Cách format USB bất kì về định dạng FAT32 (để tương thích với máy cũ, làm USB cài win, gắn máy DJ,…).

macOS

  1. Mở Disk Utility, chọn View/Show all devices
  2. Chọn USB ở cột bên trái. Ở thanh phía trên bên phải, bấm Erase.
  3. Mục 2 chọn MS-DOS (FAT). Mục 3 chọn GUID Partition Table (GPT). Bấm Erase.

Windows

Có 2 cách: Dùng Command Prompt hoặc dùng phần mềm ngoài.

Command Prompt

  1. Mở Command Prompt bằng quyền Administrator.
  2. Gõ lệnh: diskpart
  3. Gõ lệnh: list disk
  4. Nhìn xem USB là ổ đĩa số mấy. Ví dụ USB là Disk 2, gõ lệnh: select disk 2
  5. Gõ lệnh: clean
  6. Gõ lệnh: create partition primary
  7. Gõ lệnh: select partition 1
  8. Gõ lệnh: format fs=fat32 quick
  9. Gõ lệnh: active
  10. Gõ lệnh: exit

Dùng phần mềm ngoài

  1. Tải phần mềm Minitool Partition Wizard
  2. Cài đặt. Nhớ KHÔNG chọn trial bản Pro, và chỉ cài Partition Wizard, bỏ check không cài các phần mềm khác (ví dụ Shadow Recovery). Sau đó mở phần mềm lên.
  3. Bấm chuột phải vào tên USB, chọn Convert MBR Disk to GPT Disk. Mục tiêu là sử dụng GPT, nếu nút đó ghi convert GPT to MBR thì không bấm.
  4. Bấm chuột phải vào tên USB, chọn Delete all partitions
  5. Bấm chuột phải vào phần Unallocated của USB, chọn Create. Tại hộp thoại mở ra, mục File System chọn FAT32, rồi bấm OK.
  6. Tại góc dưới bên trái màn hình, bấm nút Apply, đợi cho phần mềm chạy xong.
