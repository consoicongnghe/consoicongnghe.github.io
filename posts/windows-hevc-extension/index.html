[Windows] HEVC Video Extension

Vì lí do bản quyền, Windows 10+ sau khi cài sạch mặc định không mở được một số định dạng file như HEIF, HEIC, HEVC,... khi mở sẽ hiển thị lỗi yêu cầu cài đặt thêm phần mở rộng (extension) có trả phí dạng như thế này:

Cách xử lí: Cài đặt các extension còn thiếu.

  • HEVC Video Extension giúp đọc các file ảnh định dạng HEIC (Apple), xem các file video HEVC.
  • HEIF Video Extension giúp đọc các file ảnh định dạng HEIF.
  • VP9 Video Extensions giúp đọc các file video VP9 (Google).
  • AV1 Video Extension giúp đọc các file video AV1 (Google).
  • MPEG-2 Video Extension giúp đọc các MPEG-2 (Apple).
  • Raw Image Extension giúp đọc các file ảnh định dạng raw.
  • Webp Image Extensions giúp đọc các file ảnh định dạng WEBP.
  • Web Media Extensions giúp đọc các file OGG, codec Vorbis và Theora.
