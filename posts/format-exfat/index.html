Format USB, ổ cứng định dạng exFAT

Cách format USB, ổ cứng sang định dạng exFAT (để tương thích với cả hệ điều hành Windows và macOS)

macOS

  1. Mở Disk Utility, chọn View/Show all devices
  2. Chọn USB/ổ cứng ở cột bên trái. Ở thanh phía trên bên phải, bấm Erase.
  3. Mục 2 chọn exFAT. Mục 3 chọn GUID Partition Table (GPT). Bấm Erase.

Windows

  1. Mở This PC.
  2. Bấm chuột phải vào USB/ổ cứng, chọn Format....
  3. Ở mục File system, chọn exFAT.
  4. Check ô Quick Format, sau đó bấm Start, đợi chạy xong là được.
