Hướng dẫn cài đặt Acrobat trên Windows

  1. Tải bộ cài Adobe Acrobat trên mạng. Ví dụ trang tải từ taiwebs.com

https://taiwebs.com/windows/download-adobe-acrobat-pro-dc-458.html

  1. Tắt tạm thời phần mềm chặn virus, do chúng có thể chặn nhầm phần bẻ khóa thành virus. Đối với Windows Defender, tìm “virus” trong Start menu để mở Virus & threat protection, bấm vào Manage settings. Tắt mục Real-time Protection.
  2. Chạy autoplay.exe (hoặc Setup.exe). Bấm Install, bấm Next, đợi tới khi cài xong.
  3. Mở Start Menu, gõ tìm kiếm “powershell”. Chuột phải vào Windows Powershell, chọn “Run as administrator”.
  4. Dán cụm này vào Powershell (bằng cách bấm Ctrl+C copy rồi chuyển sang Windows Powershell bấm Ctrl+V):

Stop-Process -Name "Adobe Desktop Service" -force

Sau đó bấm Enter 1 lần, rồi đóng Powershell.

  1. Mở Start menu, gõ tìm kiếm “notepad”. Chuột phải vào Notepad, chọn “Run as administrator”.
  2. Mở menu File/Open. Trong cửa sổ thư mục hiện ra, di chuyển tới C:/Windows/system32/drivers/etc/host. Tại mục chọn ở góc dưới bên phải, bấm vào chọn All files. Mở file tên là “host”.
  3. Vào trình duyệt, mở link này: https://a.dove.isdumb.one/list.txt. Bấm Ctrl+A để chọn tất cả, sau đó Ctrl+C để Copy.
  4. Quay lại phần mềm Notepad đã mở ở bước 6. Di chuyển con trỏ tới cuối file, bấm Ctrl+P để dán nội dung mới copy ở bước 8 vào file host. Sau đó bấm Ctrl+S (Save) để lưu file.
  5. Mở Adobe Acrobat và tận hưởng thành quả.
